Dokumentasi 30 September 2012
System requirements dan software requirements telah ditentukan dan difloorkan oleh analis.

KBLog adalah aplikasi yang memudahkan para wanita untuk mengingat serta memberikan informasi tentang masalah pribadi kewanitaan (masa menstruasi, masa subur, dll). Pada akhirnya dengan aplikasi ini dapat mengontrol kehamilan dan masalah kewanitaan lainnya.

Penyajian aplikasi pada program ini diusahakan seakurat mungkin terhadap data 

Penjelasan Aplikasi KBLog:
Karena aplikasi ini berisi informasi masalah pribadi dan sangat intim, maka tingkat keamanan data sangat dijaga. Dalam handset yang digunakan, terdapat satu username dan password untuk user. selanjutnya user akan masuk ke menu-menu dalam aplikasi KBLog  
Menu KBLog :
Menu Calender : berisi tentang kalender bulanan yang mencakup tanggal. Tanggal dapat dipilih sesuai dengan tanggal haid dimulai dan tanggal haid berhenti. Setelah tanggal berhenti dipilih, selanjutnya akan diinfokan masa-masa subur bagi wanita. 
Menu About : menjelaskan mengenai versi software, developer software, dll
Menu Setting : berisi tentang pengaturan password dan setting keseluruhan aplikasi.

software yang di gunakan :

android SDK 2.2 - 4.0.3
