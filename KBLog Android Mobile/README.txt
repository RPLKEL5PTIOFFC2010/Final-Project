> Deskripsi umum perangkat lunak KBlog 
     Yang akan dibuat pada project ini adalah aplikasi KBlog, yaitu suatu aplikasi yang menghitung tanggal periode dan siklus menstruasi tanpa harus mengingat-ingat atau mencatat secara manual. Pada program ini juga terdapat juga menerima masukan dari pengguna (user) berupa data pribadi yang menyangkut tanggal menstruasi, golongan darah suami / istri, tanggal lahir suami / istri, dan siklus menstruasi. Pada aplikasi ini juga terdapat Ovulation and Fertility Forecast, yaitu untuk memperkirakan tingkat kesuburan dan saat-saat subur dari seorang istri. Pada program ini terdapat Password Protection, yang mana ini untuk menjaga keamanan dan hanya user yang berkepentingan saja yang bisa membuka dan melihat aplikasi ini. 
Notification berisi hal-hal seputar Menstruasi, Pil dan alat Kontrasepsi, Pil Multivitamin, dan lain sebagainya. 
Pada aplikasi ini juga terdapat Note untuk memberi atau menuliskan catatan yang dimasukkan oleh user yang berhubungan dengan hal-hal siklus menstruasi user.

> Panduan Instalasi KBlog
1.Membuka halaman blog Official KBlog di http://rumahrpl.blogspot.com 
2.Buka halaman download Aplikasi KBlog , klik link Download KBlog 
3.Kemudian dialog untuk menyimpan muncul, pilih Save
4.Proses download akan berjalan tunggu hingga proses download selesai
5.Pilih Open untuk melakukan instalasi langsung, konfirmasi akan muncul kemudian pilih Install.
6.Tunggu proses instalasi aplikasi sampai selesai
7.Setelah proses installasi selesai maka pada handphone anda akan tampil Icon dari aplikasi KBlog
8.Pilih icon  KBlog untuk menjalankan aplikasi
