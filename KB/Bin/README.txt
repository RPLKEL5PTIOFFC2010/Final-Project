SOFTWARE REQUIREMENTS

a) Aplikasi KBlog hanya dapat digunakan pada platform Android versi 2.2 (Froyo) sampai 4.1 (Jellybean)
b) Memory yang digunakan tergantung dengan versi OS Android yang digunakan
c) Aplikasi hanya dapat dipakai oleh satu user saja
d) Aplikasi akan  terus berjalan sesuai dengan tanggal yang ada
e) Keamanan aplikasi harus didukung dengan password pada aplikasi
f) Storage yang digunakan maksimal sebesar 2MB
